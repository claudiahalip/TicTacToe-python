## TIC TAC TOE with python

a CLI Tic Tac Toe build with python that allows you to play the game 

## Instalation

install python3

use the package manager [ [pip3 ] 
to install pytest or pytest-watch

```pip3 install pytest```

```pip3 install pytest-watch```

## Starting the game

```python3 start```

## Running the tests

```ptw```

```pytest```
